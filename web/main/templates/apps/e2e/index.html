E2E Application Parameters

e2e : End to End entanglement generation between any two nodes

Sender : Source node in the network where entanglement request starts

Receiver : Destination node in the network where entanglement request ends

startTime : The request arrival time

Size : No.of entanglements requested

Priority : Priority of the request

targetFidelity: The minimum required fidelity of the requested entanglements

timeout : From startTime the time window within which entanglements need to be generated
